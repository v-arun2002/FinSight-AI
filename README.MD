## What it does

Type a ticker. Ask a question. Get a full investment brief.

FinSight pulls live stock fundamentals, Federal Reserve macroeconomic data, and real-time news вҖ” then runs LLM reasoning across all three sources to return a structured recommendation with confidence score and risk assessment.
NVDA вҶ’ "Is this a good buy given AI tailwinds?"

вҶ’ Fed funds rate:  3.63%    |  GDP growth: 2.1%
вҶ’ News sentiment:  POSITIVE
вҶ’ Analysis:        NVIDIA's dominant AI infrastructure position
justifies premium valuation despite macro headwinds...

## Why it's interesting

Most RAG and LLM demos work on static documents. FinSight runs on **live data** вҖ” every request pulls fresh prices, the latest Fed figures, and breaking news. The brief you get at 9am is different from the one at 4pm.

The pipeline is also production-grade, not a notebook:

- **Conditional routing** вҖ” if market data fails, the pipeline routes to an error handler and returns a clean response rather than crashing
- **Typed data contracts** вҖ” every component output is validated against a Pydantic schema before passing downstream, so bad data fails loudly at the source
- **Containerized and deployed** вҖ” Docker image, CI/CD via GitHub вҶ’ Render, auto-deploys on every push
- **Real API** вҖ” not a Streamlit demo, a proper FastAPI service with documented endpoints

```bash

  -e TAVILY_API_KEY=your_key \
```
